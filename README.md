# smart-file-organizer

I made this file organizer specifically to organize my projects assets folder. But I added an additional "Documents" sorting method, that could be used generally to organize document folders for any office setting.



NOTE: You do not need to use the "Organize folder by name" button. I was kind of testing, and forgot to pull that out.



\# Smart File Organizer

A lightweight desktop tool that automatically organizes messy folders using customizable rules.



Smart File Organizer is a Python-based utility designed to help creators, students, and developers keep their project directories clean. With a simple, modern UI built using CustomTkinter, it lets you sort files into categories with a single click — perfect for game dev projects, downloads folders, or any workspace that gets cluttered fast.



\---



\## ✨ Features

\- Modern, clean UI powered by CustomTkinter  

\- One-click folder organization  

\- Customizable file type rules (images, audio, documents, code, etc.)  

\- Automatic folder creation  

\- Safe moves — never overwrites existing files  

\- Light/Dark mode detection  

\- Portable .exe build (no Python required)
